shouldSuspendForAd:

Called when the ad will take over the screen in order to give the application the chance to pause ongoing activities. This is in response for user interaction with the ad.

willLeaveApplicationForAd:

Called when the user interaction with the ad triggers leaving the application. This can be, for example, opening a URL in Safari or Maps or another application registered to handle the URL specified by the ad. You should save the state of the application when you get this call.

- (void)willLeaveApplicationForAd:(id)view

Parameters

view

The ad view that triggered leaving the application.

Discussion

Called when the user interaction with the ad triggers leaving the application. This can be, for example, opening a URL in Safari or Maps or another application registered to handle the URL specified by the ad. You should save the state of the application when you get this call.
